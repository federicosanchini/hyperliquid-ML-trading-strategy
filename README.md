# Hyperliquid ML Strategy Backtest Report

This repository contains a preliminary LaTeX/PDF report describing the development of a systematic trading strategy for Hyperliquid perpetual markets.

The objective of the project is to design, test, and eventually deploy a market-neutral trading strategy on a Hyperliquid vault. The strategy exploits quantitative signals generated by machine learning models on perpetual futures markets, with the goal of making trading decisions in a fully systematic and data-driven way.

The approach is market neutral and agnostic with respect to market direction: positions are selected quantitatively from the signal rankings, rather than through discretionary market views.
